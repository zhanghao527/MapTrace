# 数据万象（CI）说明 - 用于首页缩略图优化

## 一、数据万象是什么？

**数据万象（Cloud Infinite，CI）** 是腾讯云提供的数据处理服务，与对象存储 COS 深度集成。

- **作用**：在**不存储新文件**的前提下，通过 URL 参数对 COS 中的图片做实时处理
- **常见能力**：缩放、裁剪、压缩、格式转换（如 `?imageMogr2/thumbnail/216x200`）
- **首页优化用途**：地图 Nearby 标记现在会请求类似  
  `https://xxx.cos.xxx.myqcloud.com/photos/xxx.jpg?imageMogr2/thumbnail/216x200`  
  让 COS 返回约 216×200 的缩略图，而不是几 MB 的原图，减少流量和内存占用

## 二、需要开通吗？

**需要**。如果未开通，带 `imageMogr2` 参数的 URL 会返回 404 或原图，优化无效。

当前逻辑已做降级：非 COS 地址或无法识别时，会直接使用原图 URL，不会报错。

## 三、如何开通？

### 方式一：在 COS 控制台为存储桶开通（推荐）

1. 登录 [腾讯云 COS 控制台](https://console.cloud.tencent.com/cos)
2. 找到存放照片的存储桶（与 `application.yml` 中配置的 `cos.bucket` 一致）
3. 进入 **存储桶配置** → **数据处理** → **图片处理**
4. 开启「数据万象」或「图片处理」能力  
   （若页面提示需先绑定数据万象，按提示完成即可）

### 方式二：在数据万象控制台创建并绑定

1. 登录 [数据万象控制台](https://console.cloud.tencent.com/ci)
2. 选择与 COS 相同地域，创建或绑定数据万象实例
3. 将 COS 存储桶与该实例绑定

### 开通后验证

在浏览器访问：

```
你的照片URL + ?imageMogr2/thumbnail/216x200
```

- 若能正常返回缩小后的图片 → 已生效
- 若 404 或显示原图 → 检查存储桶是否已正确绑定数据万象

## 四、计费说明

- 数据万象按量计费，包含基础图片处理（缩放、裁剪等）
- 有免费额度，小程序缩略图这类低频请求一般成本很低
- 详情见：[产品定价](https://buy.cloud.tencent.com/price/ci)

## 五、若暂时不开通

- 首页仍可正常使用，只是地图 Nearby 会加载原图，体积较大
- 后续可在后端生成并保存真实缩略图（`thumbnail_url`），再移除该 URL 参数优化

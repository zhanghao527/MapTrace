# 地图时迹 - 开发工具

## 开发环境

| 类别 | 工具 | 说明 |
|------|------|------|
| IDE | Kiro + 微信开发者工具 + IDEA | Kiro 写代码，微信开发者工具预览和调试，IDEA 开发 Spring Boot 后端 |
| 小程序框架 | 原生微信小程序 | MVP 阶段使用原生开发，调试方便，后续视情况考虑跨端框架 |
| UI 组件库 | WeUI | 微信官方出品，风格统一，轻量 |
| 后端框架 | Spring Boot | Java 技术栈，自建后端，便于后续扩展社区、私信等功能 |
| 图片存储 | 腾讯云 COS | 对象存储，自带 CDN 加速 |
| 版本管理 | Git + GitHub | 仓库地址：https://github.com/lantianit/MapTrace.git |
| 设计工具 | AI 辅助设计 | 通过 AI 根据 PRD 直接生成 UI 方案 |
| 文档管理 | Markdown | 项目内 docs/ 目录，跟代码一起版本管理 |
